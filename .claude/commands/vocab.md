# 単語データ更新スキル

単語データを手動で追加するスキル。

## 手順

1. まず現在のプロジェクトにある `vocab/data.js` を読んで形式・最後のIDを把握する。

2. `$ARGUMENTS` にデータがあればそれを使う。なければ「単語データを貼り付けてください」と聞く。

3. ユーザーのデータ形式（例）：
   ```
   abandon / 捨てる、断念する / 2級
   例文: He abandoned his plan.
   訳: 彼は計画を諦めた。
   ```
   レベル対応：3級→`'3'`、準2級→`'pre2'`、2級→`'2'`、準1級→`'pre1'`

4. 手順1で確認した形式・IDの続きで WORDS 配列の末尾に追加する。

5. `git add vocab/data.js && git commit -m "vocab: add words"`

6. 何単語追加したか報告する。
